# Customer_Discount_Checker

A simple Python program that checks whether a customer is eligible for a discount based on their age.

## 📌 Project Overview

This project takes customer names and ages as input, then checks if they qualify for a **10% discount** based on the given age condition.

It is a beginner-friendly Python project that demonstrates:

- Variables
- Lists
- Loops
- Conditional statements
- `zip()` function
- Output formatting

---

## 🚀 Features

- Stores multiple customer names and ages
- Checks eligibility automatically
- Displays personalized messages
- Easy to modify for different discount rules

---

## 🛠️ Technologies Used

- Python 3

---

## 📜 Discount Rule

Customers aged between **22 and 30** are eligible for a **10% discount**.
